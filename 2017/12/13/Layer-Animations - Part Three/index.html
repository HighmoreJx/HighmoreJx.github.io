Layer Animations - Part III

发表于 2017-12-13 | 分类于 iOS ， Animation | 次

这节主要翻译一些特殊的动画。

形状与遮罩（Shapes and Masks）

动画属性 说明
path 将显示层改变成不同的形状
fillColor 将填充色改为不同的颜色
lineDashPhase 在图形周围画电线
lineWidth 图形边线宽度

梯度/渐变动画（Gradient Animations）

CALayer的坐标系如下图所示，渐变动画也是基于该坐标系进行的。

属性 说明
colors 所有梯度显示的颜色数组
locations 值递增的数组，范围在上诉坐标系范围内
startPoint 绘制起始坐标，默认（0.5, 0）
endPoint 绘制结束坐标，默认（0.5，1）
type 颜色线性变化控制，默认kCAGradientLayerAxial

路径动画（stroke and path animations）

CAShapeLayer

属性 说明
path 图形边线路径
lineWidth 边线宽
stokeColor 边线颜色
lineDashPattern 边线样式，数组，依次位实线，空白的宽度
lineDashPhase 虚线显示的其实距离，设置为2，则显示长度2之后的线
lineCap 线终点样式
lineJoin 线拐点样式
strokeStart stokeEnd 起始点，按照layer坐标系设置
fillColor 图形填充色

strokeEnd绘制，stroke擦除

重复动画（replicating animations）

CAReplicatorLayer

属性 说明
instanceCount 复制次数
instanceDelay 复制相对时间间隔
instanceTransform 复制相对位移
preservesDepth 是否开启景深
instanceColor 复制颜色
instanceRedOffset 相对红色偏移量
instanceGreenOffset 相对绿色偏移量
instanceBlueOffset 相对蓝色偏移量
instanceAlphaOffset 相对透明偏移量
